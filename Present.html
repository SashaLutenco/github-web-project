Present

CSS3 is the current version for CSS. There are speculations of CSS4 being in development, but as of 2020 CSS3 is still in effect. These are some of the new features that have been added onto CSS3:

grid-gap, column, custom-properties, box-alignment, @supports, scroll-snap, clip-path, mask, vw, vh, vmax, vmin, min-height: 100vh, caret-color:orange, ch units, CSS Angles - deg(), grad(), rad(), turn(), Conic (cone) Gradients

CSS3 has made plenty of alterations, so to save time I'll focus primarily on colors. CSS3 has added onto RGBA, added colors, gradients, changes in opacity, shadows. It added rounded borders, and the list goes on and on.
