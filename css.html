“CSS or Cascading Style Sheets is a design language that has the function to simplify the process of making web pages look more aesthetic. CSS allows developers to mold the look of the webpages to their needs. It is in control of the color of the text, the style of fonts, the spacing between paragraphs, how columns are sized and laid out, what background images or colors are used, layout designs, variations in display for different devices and screen sizes as well as a variety of other effects. CSS is used to define styles for your web pages, including the design, layout and variations in. The style definitions are normally saved in external .css files.”

  • CSS saves time : With CSS you can apply the same code for as many different HTML pages. Pages load faster – CSS allows developers to reduce the use of attributes which benefits to not have as many files loading.

  • Easy maintenance: Easier to make changes in all the pages by only adjusting some in the CSS file.

  • Superior style to HTML: CSS has a bigger advantage over HTML because of the styling tools available.

  • Multiple Device Compatibility : CSS can be modified to be used and presented on all kind of devices.

  • Global web standards: It is being recommended to use CSS in order to stay compatable with future browsers.
